###
otlcircuit.R: OTL Circuit function https://www.sfu.ca/~ssurjano/otlcircuit.html

test_otlcircuit.R: Example for the OTL Circuit function with original input dimension d = 6, training set size n = 200, and test set size m = 1000.

result_otlcircuit_200.Rmd: Process the MCMC posterior samples from "test_otlcircuit.R" to obtain  standardized RMSE and boxplots.

test_otlcircuit_500.R: Example for the OTL Circuit function with original input dimension d = 6, training set size n = 500, and test set size m = 1000.

result_otlcircuit_500.Rmd: Process the MCMC posterior samples from "test_otlcircuit_500.R" to obtain  standardized RMSE and boxplots.

test_otlcircuit_20dim.R: Example for the OTL Circuit function with increased input dimension d = 20, training set size n = 200, and test set size m = 1000.

result_otlcircuit_200_20.Rmd: Process the MCMC posterior samples from "test_otlcircuit_20dim.R" to obtain  standardized RMSE and boxplots.

test_otlcircuit_500_20dim.R: Example for the OTL Circuit function with increased input dimension d = 20, training set size n = 500, and test set size m = 1000.

result_otlcircuit_500_20.Rmd: Process the MCMC posterior samples from "test_otlcircuit_500_20dim.R" to obtain  standardized RMSE and boxplots.
