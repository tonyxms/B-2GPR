###
piston.R: Piston simulation function https://www.sfu.ca/~ssurjano/piston.html

test_piston.R: Example for the Piston simulation function with original input dimension d = 7, training set size n = 200, and test set size m = 1000.

result_piston_200.Rmd: Process the MCMC posterior samples from "test_piston.R" to obtain standardized RMSE and boxplots.

test_piston_500.R: Example for the Piston simulation function with original input dimension d = 7, training set size n = 500, and test set size m = 1000.

result_piston_500.Rmd: Process the MCMC posterior samples from "test_piston_500.R" to obtain standardized RMSE and boxplots.

test_piston_20dim.R: Example for the Piston simulation function with increased input dimension d = 20, training set size n = 200, and test set size m = 1000.

result_piston_200_20.Rmd: Process the MCMC posterior samples from "test_piston_20dim.R" to obtain standardized RMSE and boxplots.

test_piston_500_20dim.R: Example for the Piston simulation function with increased input dimension d = 20, training set size n = 500, and test set size m = 1000.

result_piston_500_20.Rmd: Process the MCMC posterior samples from "test_piston_500_20dim.R" to obtain standardized RMSE and boxplots.
